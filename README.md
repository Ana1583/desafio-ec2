# Laboratório: Gerenciamento de Instâncias EC2

**Resumo**
Este repositório contém anotações, comandos e resultados do laboratório sobre gerenciamento de instâncias EC2 na AWS. O objetivo foi praticar criação, configuração, acesso e monitoramento de instâncias EC2.

---

## Conteúdo do repositório
- `README.md` — arquivo principal com resumo e instruções de validação.
- `notes.md` — anotações detalhadas do que foi realizado e aprendizados.
- `commands.md` — comandos prontos para copiar/colar (SSH, EBS, mount, git).
- `/images` — pasta opcional com capturas de tela relevantes.

---

## Objetivos do laboratório
- Criar instâncias EC2 (Amazon Linux 2 ou Ubuntu)
- Configurar chaves SSH e Security Groups
- Conectar via SSH e executar tarefas básicas
- Anexar e montar volumes EBS
- Criar snapshots e entender backups
- Monitorar métricas com CloudWatch
